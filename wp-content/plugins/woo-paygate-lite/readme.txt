WooCommerce插件微信支付、支付宝支付网关

== Description ==

Woo PayGate Lite 插件可为 WooCommerce 插件提供微信支付、支付宝支付接口，实现在线支付功能。

如需支持手机端支付可<a href="https://www.wpcom.cn/plugins/woo-paygate.html" target="_blank">升级付费版</a>。


== Installation ==

1. 直接在wordpress后台搜索插件安装并启用；
2. 下载插件并解压，上传至 `/wp-content/plugins/` 目录，然后启用插件；
3. 进入wordpress后台【WooCommerce>设置>付款】下设置支付网关
